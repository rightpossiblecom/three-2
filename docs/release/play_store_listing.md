# Three. - Play Store Listing

## Short Description
A minimal daily focus app. Choose your three most important tasks for the day.

## Full Description
Three. is a minimal, distraction-free daily focus app. 

Instead of an endless to-do list that causes overwhelm, Three. challenges you to pick exactly three tasks that matter most today. At the start of your day, add up to three tasks. When you complete them, check them off to build your daily streak. 

Key Features:
- Add up to 3 crucial tasks per day, forcing intentional prioritization.
- A distraction-free, minimalist user interface focusing only on what matters.
- Daily streaks and statistics keep you motivated.
- Tasks automatically reset at midnight.
- 100% offline and private. No user accounts, no cloud syncing, your data stays entirely on your device.

Start your day with intent. Focus on Three.
